C# SM2加解密工具：本项目旨在用国密SM2进行加解密数据的C#调用示例。作者是一名全栈工程师，该项目之前用于验证C++平台与终端通信的SM2加密报文做的小工具，现将其开源希望给大家提供做程序思路，如果想做成库进行引用只需将工程做成类库，进行调用即可，若有疑问可留言...
##SM2加解密工具功能说明
SM2加解密工具共有三个功能，分别是随机生成SM2密钥对、加解密SM2数据和文件解密密文
